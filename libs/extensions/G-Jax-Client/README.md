**G-Jax-Client** is an **UI** add-on for the projects **[Jax-Y]( https://github.com/rac021/Jax-Y)**  and **[Jax-D]( https://github.com/rac021/Jax-D)** 

## installation

```xml
  mvn clean compile assembly:single
```  
```xml
  java -jar target/g-jax-client.jar
```
